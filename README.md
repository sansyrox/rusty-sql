# Rusty Robyn SQL

This is a simple Robyn app using a RUST SQL connection.

## Usage
Install Robyn
```bash
$ pip install robyn
```

Run the app
```bash
$ python -m robyn app.py --compile-rust-path="."
```

Run the app in dev(live reloading) mode
```bash
$ python -m robyn app.py --compile-rust-path="." --dev
```
